# Asset Simulator Feedback

Welcome to the feedback and issue tracking repository for the **Asset Simulator**. 

This repository is dedicated strictly to collecting feedback, reporting bugs, and suggesting new features from our trusted testers. 

---

### 🚀 [Launch the Asset Simulator](https://asset-simulator-zeta.vercel.app/)

---

## How to provide feedback

We value your input! To ensure your feedback is captured and triaged correctly, please use the following links:

*   **[🐛 Report a Bug](https://github.com/jricketson/asset-simulator-feedback/issues/new?template=bug_report.md)** – Something isn't working right or the math looks off? Let us know.
*   **[✨ Request a Feature](https://github.com/jricketson/asset-simulator-feedback/issues/new?template=feature_request.md)** – Have an idea for a new strategy, asset type, or visualization? We'd love to hear it.

## Guidelines
*   **Privacy First**: Please **never** post real personal financial data (PII), account numbers, or sensitive identity information in public issues. Use rounded numbers and illustrative scenarios.
*   **Be Descriptive**: The more detail you provide (including screenshots and steps to reproduce), the faster we can fix issues.
*   **Check Existing Issues**: Before opening a new issue, please check if someone else has already reported it.

## Project Goal
The Asset Simulator is designed to help Australians (and expats) model complex financial scenarios—including Superannuation, Family Trusts, and Investment Loans—with high precision and mathematical transparency.

Thank you for being part of our trusted tester group!
